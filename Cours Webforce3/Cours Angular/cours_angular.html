ANGULAR

il est existe de angulars : ANGULAR, et ANGULAR JS. Ces 2 version sont totalement différentes

framework = fournir un cadre de travail

angular utilise le typescript, c'est un framework JavaScript coté client, il est orienté asynchrone (permet de modifier la page sans la rafraichir entièrement)

type de script : number, booleens, tableaux, objet, undefined, string, flotant...

pour que le navigateur comprenne le langage ANGULAR on va devoir utilisier la Transpilation, il s'utilise souvent avec IONIC (pour créer des aplication mobile.

pour installer ANGULAR on doit installer NPM (node package manager), c'est un gestionnaire de pacquets (code source)

quand on fait du ANGULAR on fait du composant (script, html et css)

LES CLASSES

a partir des classes (ensemble de champs et de méthodes) on va pouvoir définir des objets, elles peuvent être modifier pour en faire un composant

LES MODULES

un regroupement de plusieurs composant
