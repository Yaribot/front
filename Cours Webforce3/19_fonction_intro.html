Les instructions

un script js est un fichier contenant des instructions, placée les unes à la suite des autres. chacune des ces instructions peut-être vue comme un ordre donné au moteur javascript, une action que vous souhaitez réaliser, à la manière d'une recette de cuisine

  • faire fondre le beurre
  • Mélanger le beurre et la farine
  • Verser le chocolat petit à petit etc.

Les instructions se suivent et sont effectuées les unes à la suite des autres, Du haut vers le bas (comme en CSS), le browser parse (analyse) dans l'ordre du flux

Chaque instruction DOIT se terminer par point virgule ; afin d'indiquer au moteur Javascript que cette instruction est terminée

Les fonctions

les fonctions permettent de regrouper une série d'instructions pour pouvoir les réutiliser. Quand on a besoin du code d'une série d'instructions, plutôt que de réécrire à chaque fois toutes les instructions, on appellera une fonction par son nom (faireUnGateau). Cette donction contient l'ensemble des instructions à exécuter.

Par exemple la fonction alert est une fonction native (intégrée au navigateur) qui contient en gros les instructions suivante:

  • Créer le cadre dans la boite de dialogue
  • Positionner le cadre dans la fenêtre
  • Créer un background semi transparant sur toute la page
  • Placer un bouton avec le texte OK et un événement click
  • Créer une zone de texte qui affichera le texte de l'art à présenter au visiteur

Nous on a juste besoin d'utiliser la fonction alert() en lui donnant un argument exemple alert('coucou'). La fonction éxécutera toutes les instructions à chaque fois.
On peut utiliser cette fonction alert quand on veut en la déclanchant avec alert (quelque chose). Ce "Quelque chose" est un ARGUMENT attendu par la fonction.

var maVariable = document.querySelector('h2'); alert('Coucou')

Les fonctions querySelector et alert sont disponibles dans le navigateur, ce sont des méthodes

Si vous voyez quelque chose qui ressemble à une variable et qui est suivie de parenthèse c'est une fonction.

Les fonctions prennent souvent des arguments - des bouts de données dont elles ont besoin pour faire leurs travail. kes argments sont placé ENTRE les parenthèses, séparées par des virgules s'il y en a plusieurs.

Par exemple la fonction alert() fait apparaitre une popup mais vous devez donner une chaine comme argument pour indiquer à la fonction ce que vous souhaitez écrire dans la fenêtre.

Nous pouvons écrire nos propres fonctions
