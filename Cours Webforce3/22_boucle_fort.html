Les boucles

Les instructions for et while qui sont deux types de boucles les plus utilisées e js, servent à repérer plusieurs lignes de code

La boucle for

for prend 3 paramètres encadrés dans des parenthèses, suivi entre accolades des instructions à éxécuter autant de fois que la boucle l'indique. les paramètres sont :

  • Initialisation : Ici on déclare la variable qui servira de compteur
  • Condition : Tant que cette condition est vraie la boucle s'éxécutera
  • Expression final : Expression qui est éxécutée à chaque fin de boucle
for(initialisation; condition; incrémentation){ instruction 1; instruction 2; }
