JQuery, c'est une bibliothèque JavaScript, ou un framework en anglais. C'est une boite à outils de blocs de code réutilisable. cette librairie est rapide, petite donc légère et riche en fonctionnalités.

Elle simplifie considérablement la navigation et la manipulation dans le DOM des pages HTML, la gestion des événements, l'animation et Ajax (Asynchronous Javascript And XML) grâce à une API (Application Programing Interface) facile à utiliser et qui est pré testée pour nous et qui fonctionne sur une multitude de navigateurs.

JQuery est elle-même codée en Javascritp

Polyvalence et extensibilité, Jquery a changé la façon de travailler de millions de développeurs.

  • Fonctions toutes faites, optimisées, qu'on utilisera souvent : appel Ajax, boucles, animations, toggle
  • résoud beaucoup de problèmes de compatibilité navigateurs. Jquery à des polyfills intégrés
  • Une lécture de code plus simple car les problèmes sont géré par la librairie

Sélectionner en JQuery

Le coeur de JQuery se trouve dans la sélection déléments pour ensuite éxécuter des méthodes dessus.

$(selecteur) accepte en argument tout sélecteur CSS, JQuery nous renvoie ensuite une collection d'un ou plusieurs éléments DOM enveloppés dans un objet JQuery.

$('div') sélectionne toutes les divs $('.titre') sélectionne touts les éléments qui ont la class='titre' $('#boite') sélectionne uniquement les éléments qui dispose de l'ID boite

Cette enveloppe JQuery est celle qui donne accès à toutes les fonctionnalités de JQuery

  • Rajouter des classes
  • Modifier le contenu HTML
  • Afficher ou cacher des éléments
  • Les animer etc...

Tous les éléments sélectionnés en JavaScript pur (document.getElementById, querySelector etc...) sont des éléments DOM dans une enveloppe JQuery

JQuery dipose de plusieurs méthodes pour parcourir l'arbre du document (le DOM)

  • children() parcourt les enfants
  • find() parcours les déscendants
  • next() renvoie le voisin suivant
  • prev() renvoie le voisin précédent
  • parent() renvoie, parmi une liste d'éléments, celui qui se trouve à l'index fourni
